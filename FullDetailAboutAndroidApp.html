Android Application Development

Java, Kotlin, Ionic, Dart (Flutter SDK), Xamarin

Mobile application development have help to improve privacy, security and easy access while using internet.
Our training on Android Application Development Involves the following Programming Language and students are allowed to picked only one language out of these:
Java android (core android language),
Kotlin,
Javascript (Ionic ),
Dart ( Flutter sdk),
C# (Xamarin).
Certainly, interested student will be working with Android Studio IDE (Integrated Development Environment).
The use of Firebase, Sqlite, Admob, Gson , Json and Google play store will be explore during the training of this program.
Api intergration (Google Api, Payment Api, Amazon Api, Facebook Api, other third party Api )
will be treated as well.
